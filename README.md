📌 Overview
End-to-end Sales Analytics project leveraging SQL and Python to analyze revenue trends, regional performance, and profitability, delivering actionable business insights through data visualization.

## 🛠 Tools Used
- Python (Pandas, SQLite)
- Google Colab
- Matplotlib

## 📊 Key Features
- Data Cleaning and Preprocessing
- SQL-based Analysis
- Data Visualization
- Profit Margin Analysis

## 📈 Key Insights
- Identified top-performing regions and products
- Analyzed seasonal sales trends
- Detected loss-making categories
- Evaluated profit margins across sub-categories

## 📂 Files
- `Sales_Analytics_Project.ipynb` – Main notebook
- `superstore.csv` – Dataset

## 🚀 Outcome
Provided actionable insights to improve business performance and profitability.
